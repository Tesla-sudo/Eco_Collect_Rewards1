# Eco_Collect_Rewards1

# 1. Backend Setup
Installation of dependencies 
cd Backend

npm install express mongoose dotenv cors openai morgan jsonwebtoken joi helmet axios compression bcryptjs @sendgrid/mail @google/generative-ai @google/genai express-rate-limit debug 

npm install --save-dev nodemon

npm run dev


# 2. Frontend setup
cd client

npm install axios react react-dom react-router-dom tailwindcss

npm run dev
